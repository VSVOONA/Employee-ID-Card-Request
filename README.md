# Employee ID Card Request

A ServiceNow project that automates the Employee ID Card Request process using **Service Catalog** and **Flow Designer**.

## 📌 Overview

The Employee ID Card Request System allows employees to submit requests for a new or replacement ID card through a Service Catalog item. The system automates the complete workflow by routing requests for approval, sending email notifications after approval, and creating a Catalog Task for the administration team.

## ✨ Features

- Employee ID Card Request Catalog Item
- Employee Name, Department, and Reason fields
- Automated Approval Workflow
- Email Notification after Approval
- Automatic Catalog Task Creation
- End-to-End Workflow Automation
- Request Tracking using ServiceNow

## 🛠️ Technologies Used

- ServiceNow
- Service Catalog
- Flow Designer
- Catalog Variables
- Approval Actions
- Email Notifications
- Catalog Tasks

## 🔄 Workflow

1. Employee submits an ID Card Request.
2. Flow Designer retrieves catalog variables.
3. Request is sent for approval.
4. Upon approval, an email notification is sent.
5. A Catalog Task is created for the administration team.
